# Перемещение максимального элемента в начало списка

## Задача

Заполнить список случайными числами в диапазоне [10, 99], найти максимальный элемент и переместить его в начало списка.

## Пример работы

Исходный список: [23, 67, 45, 89, 12, 78, 34, 56, 91, 28]
Список после перемещения: [91, 23, 67, 45, 89, 12, 78, 34, 56, 28]
