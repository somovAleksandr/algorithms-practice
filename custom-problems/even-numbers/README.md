# Вывод чётных чисел

## Задача

Написать программу, которая выводит только чётные числа из диапазона от 1 до 20.

## Пример

Input: Диапазон от 1 до 20  
Output:
i = 2
i = 4
i = 6
i = 8
i = 10
i = 12
i = 14
i = 16
i = 18
i = 20

## Тестирование

Для запуска тестов выполните:

```bash
python test_even_numbers.py
```
